Projects

These are some of my projects that I'm most proud of. These usually have some niche utility and are mostly tools for Android. I have additional projects, though not as interesting, in my GitHub.

notifier.sh

  • Full-stack application with Web, Android, and back-end components.
  • Tool to receive notifications on an Android phone when a long-running terminal process finishes.
  • Users can see previous notifications and usage details on the app and on the web.

BetterFilter

  • Content filter and adblocker for Android, aimed primarily at parents to wish to safeguard their children for inappropriate material.

WhatsappGrouper

  • Server application that attempts to combine WhatsApp groups together to enable having more than 257 people on a single chat.>
  • Integrates with chat-api.com's REST API
  • This project was successfully used, but had to be retired because the API doesn't support inline replying, which results in a very messy and confusing experience.
